Breadbox is an open-source, self-hosted financial aggregator for households. It syncs your bank transactions into a PostgreSQL database you own and exposes them through a REST API and a built-in MCP server for AI agents.

## Run Breadbox in three commands

Spin up Breadbox and PostgreSQL with Docker Compose:

```bash
curl -O https://raw.githubusercontent.com/canalesb93/breadbox/main/docker-compose.yml
cp .env.example .env
docker compose up -d breadbox db
```

Once the containers are healthy, open your Breadbox URL — `http://localhost:8080` for local test runs — and the first-run wizard will walk you through creating your admin account, configuring a bank provider, and setting your sync interval.

<Tip>
  `http://localhost:8080` is fine for kicking the tires, but most Breadbox instances live on a publicly reachable host so agents, webhooks, and family members can reach them. See [Remote Hosting](/installation/remote-hosting) for production setups.
</Tip>
